# Context

Drop files here that Claude should use when building your design system and slides.

## For Design Setup

Put anything that represents your visual identity:

- **Brand guidelines** (PDF, images) — logos, colors, typography specs
- **Inspiration** — screenshots of decks, websites, or magazines you like the look of
- **Existing materials** — your current slides, pitch deck, one-pager, investor memo
- **Competitor decks** — what you want to look different from (or similar to)
- **Reference URLs** — save them in a text file (e.g., `inspiration-urls.txt`)

Claude reads everything in this directory when running the Design Setup. The more you give it, the better your design system will be.

## For Deck Building

As you build slides, use this directory for:

- **Research** — market data, reports, articles that inform your content
- **Notes** — meeting notes, brainstorm docs, narrative outlines
- **Source data** — spreadsheets, charts, metrics for data slides

## File Types

Claude can read: PDF, PNG, JPG, MD, TXT, CSV, and most text-based formats.
